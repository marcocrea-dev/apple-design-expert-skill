## Purpose

- [ ] Is there a clear primary action on every screen?
- [ ] Does every visible feature serve a core user goal?
- [ ] Has anything been intentionally excluded to preserve focus?
- [ ] Can a new user understand the app's purpose within 5 seconds?
- [ ] Are notifications meaningful and contextual, not promotional?

## Agency

- [ ] Can every destructive action be undone?
- [ ] Are destructive actions confirmed before execution?
- [ ] Can users navigate freely, or are they forced into linear flows?
- [ ] Are there sensible defaults with easy overrides?
- [ ] Does the user feel in control, not guided?

## Responsibility

- [ ] Are permissions requested in context, not at first launch?
- [ ] Are purpose strings clear and specific?
- [ ] Is data collection minimized and explained?
- [ ] Are AI features labeled and their outputs explainable?
- [ ] Is there a manual override for every automated action?

## Familiarity

- [ ] Do system components look and behave as expected on this platform?
- [ ] Do similar-looking elements behave consistently?
- [ ] Are SF Symbols used for icons where applicable?
- [ ] Do metaphors map cleanly without being too literal or abstract?
- [ ] Is navigation consistent across all screens?

## Flexibility

- [ ] Does the layout work on all supported screen sizes?
- [ ] Can all actions be completed with keyboard, voice, and touch?
- [ ] Is Dynamic Type supported with no layout breakage?
- [ ] Are touch targets at least 44x44 points?
- [ ] Can users customize or reorder controls?

## Simplicity

- [ ] Is there exactly one primary action per screen?
- [ ] Is copy written in plain language (8th-grade reading level)?
- [ ] Is visual hierarchy clear (size, weight, color, spacing)?
- [ ] Are advanced options progressively disclosed?
- [ ] Would removing any element harm the experience?

## Craft

- [ ] Are empty, loading, error, and success states designed?
- [ ] Do animations feel responsive (under 300ms) and respect reduced motion?
- [ ] Is typography readable at all Dynamic Type sizes?
- [ ] Does the app feel polished on the oldest supported device?
- [ ] Are there dropped frames, layout shifts, or jank?

## Delight

- [ ] Does the experience create a positive emotional response?
- [ ] Are micro-interactions meaningful and purposeful?
- [ ] Is the copy human, not robotic?
- [ ] Is delight earned through solving problems, not decoration?
- [ ] Can all delightful elements be dismissed or reduced by the user?
